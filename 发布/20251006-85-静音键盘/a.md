# **静音** 键盘简单评测

**敲键盘声音太大怎么办 !?**
老公今年确诊糖尿病, 不能熬夜, 晚上要睡好.
窝干日结临时工, 每天下班回来就比较晚了, 但不想虚度光阴
(上班挣钱出卖劳动力只是为了 "活着", 除了挣钱毫无意义),
所以经常敲键盘写点东西.
在北京偏远农村, 租的小房间面积很小 (为了降低房租),
所以老公在旁边睡, 窝在旁边 "啪啪啪" (敲键盘) 距离很近.
问题是, 敲键盘声音太大, 吵的老公睡不着 !!

窝经常会敲键盘到 0 点左右 (北京时间), 所以对键盘 **静音** 有很大需求.
别的可以不管, 但是必须 **静音 ! 静音 !! 静音 !! !**

## 1 评测方法

对每个键盘, 评测内容包含:

+ **外观**: 通过拍照片展示键盘的外观.

+ **噪声**: 模拟使用 (敲击键盘), 同时使用仪器测量噪声的分贝大小.

+ **主观感受**: 使用键盘的主观感受, 仅供参考.


## 2 评测过程及结果

测试时间: 深夜 (北京时间 23 点)

室内背景噪声: `25dB`

测试的键盘包括: **薄膜键盘** (最常见的普通键盘), 静电容键盘, 硅胶软键盘.

没有 `机械键盘`, 因为从经验来看, 机械键盘的噪声是比较大的.

### 2.1 普通薄膜键盘 (黑, HP)

![淘宝 (1)](./图/21-t-1.png)

图片来源: 淘宝 app 截图 (下同)

这个就是最普通的薄膜键盘, 比较便宜. 能用, 没有明显特点.

![键盘 (1)](./图/21-k-2.png)

噪声: **40dB**

这个键盘是本次测试中噪声最大的 ! 可能这就是便宜的代价吧.

### 2.2 无线薄膜键盘 (白, HP CS10)

![淘宝 (2)](./图/22-t-1.png)

这是一个白色的无线键盘, 也是薄膜键盘.

![键盘 (2)](./图/22-k-2.png)

噪声: **38dB**

噪声也不小, 这个键盘在无线键盘中也算比较便宜的.

### 2.3 静电容键盘 (Niz)

![淘宝 (3)](./图/23-t-1.png)

然后就来到了传说中的 **静电容** 键盘, 也有人叫 "高级薄膜" 键盘.
虽然看起来挺贵, 但这个键盘在静电容键盘中, 也是比较便宜的.

![键盘 (3)](./图/23-k-2.png)

这个是去年送老公的礼物, 因为他经常在键盘上吃饭, 所以很脏.
选择静电容是因为, 薄膜键盘他嫌手感不好, 就要闹了.
便宜的机械键盘又不耐用, 经常损坏 (详见 《廉价机械键盘维修: 使用电烙铁更换损坏的轴》).
而静电容键盘就皮实耐造多了, 全密封防水, 没有机械磨损部件, 理论上几乎有无限寿命,
同时手感还可以, 预计可以使用 5 ~ 10 年, 这样相比机械键盘就并不贵了.

那么, 静电容键盘的噪声如何 ?

噪声: **37dB**

啊, 噪声也不小 !

## 3 总结与展望

各键盘的噪声测试结果总结如下 (室内背景噪声 25dB):

| 键盘 | 噪声 |
| :--- | ---: |
| 普通薄膜键盘 (黑, HP) | 40dB (+15) |
| 无线薄膜键盘 (白, HP CS10) | 38dB (+13) |
| 静电容键盘 (Niz) | 37dB (+12) |
| 静音薄膜键盘 (LT L1) | 35dB (+10) |
| 巧克力薄膜键盘 (白, Dell KB216) | 28dB (+3) |
| 巧克力薄膜键盘 + 键盘膜 | 27dB (+2) |
| 硅胶软键盘 (黑) | 28dB (+3) |

毫无疑问, 本次测试的各键盘中, 噪声最小的键盘是 `KB216` !
虽然这个键盘并没有强调自己是 "静音键盘", 噪声却是最小的.
同时使用起来的手感还可以, 如果有静音的需求, 可以考虑这个键盘.

本次测试的范围有限, 测量的误差较大, 主观因素较多, 结果仅供参考.
如果读者知道噪声更小的键盘, 欢迎在评论区指出哦 ~

相比之前 (38dB), 新的静音键盘 (KB216) 在噪声方面有显著改善.
希望老公晚上能睡好了 (狗头).
