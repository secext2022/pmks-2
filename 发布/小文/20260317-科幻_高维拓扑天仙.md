# (科幻) 平行宇宙修仙原理 (高维拓扑天仙)

基于这个设定的扩展 (解释):

+ 修士威压: 修士基于自身修为, 在 肉身 (世界膜) 附近制造的 时空弯曲, 根据广义相对论, 这种弯曲可以表现为 "引力". 高阶修士甚至可以制造 "黑洞" 级别的强烈时空弯曲.

+ 打坐 (入定): 修士的本体 (大部分) 在 高维泡 里, 肉身 主要是 时空坐标 锚点, 或者说像一个 "终端". 修炼时, 修士的 "意识" 从肉身终端 "下线", 前往 高维泡 进行施工. 此时在 凡世 看来, 修士就好像睡着了一样. 修炼结束, 修士重新 "上线" 肉身终端, 表现为 "醒来".

+ "内视": 修士的 "内视" 并不是看自己在 凡世 的 肉身 内部, 而是看 "高维泡" 的内部结构. 但是, 修士曾经也是凡人, 把高维泡 "比喻" 为身体内部, 方便理解, 符合习惯.

+ 经脉: "经脉" 并不是肉身上的结构, 而是 高维泡 结构的投影. 这种 "映射" 方便修士理解, 感觉比较熟悉. 还有和 "灵根" 有关: "冲开" 经脉, 其实是改造 "世界膜缺陷" 的形状, 从而能够更高效的连接肉身和高维泡.

+ 分身术: 至少分为 2 种, "低维分身术" 和 "高维分身术".

低维分身术: 技术含量较低. 修士只有一个 "本体" (肉身) 通过 世界膜缺陷 直接与 高维泡 相连. 其余 "分身" 只是 凡世 内的低维平凡结构, 在 凡世 内部与 本体 相连, 就像远程遥控机器人.

高维分身术: 一般只有高阶修士可以掌握. 修士可以 "制造" 新的 世界膜缺陷, 或者说把自己原来的 世界膜缺陷 分割开, 形成新的可独立存在的 世界膜缺陷. 修士的本体 (高维泡), 对普通修士来说, 就像高维泡只伸出 "一只脚" (触手) 连接到 世界膜 上 (肉身 锚点). 而高维分身术, 高维泡 可以同时伸出多个 "脚", 连接到 世界膜 的不同位置, 也就是说, 这种修士具有 "多个锚点". 当然, 修士可以选择其中一个 锚点, 当做 "本体", 但这不是必需的. 修士本体 (高维泡) 就像 服务器, 只有一台. 但 "终端" (高维分身 肉身) 可以有多个.

+ 修士的生育繁殖: 低阶修士可能还有少量残留的繁殖欲望. 但高阶修士, 离 凡人 更远, 离 神 (天仙) 更近, 通常不会有繁殖欲望. 因为, 和 混沌空间 的无穷维广阔相比, 这些就像小孩子玩泥巴那样低级和无聊.
