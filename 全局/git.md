# git 平台收集

能存放 git 仓库的平台 (网站).


## git 平台分布 (11)

+ 中国 (4):
  确实开源 (GitLink),
  云效代码 (codeup, 阿里云), atomgit, 码云 (gitee)

+ 美国 (1): github

+ 欧洲 (2): codeberg, framagit

+ 澳大利亚 (1): bitbucket

+ 未知 (3): notabug, disroot, pub.solar


## 中国

+ `www.gitlink.org.cn` (确实开源)

  可存放开源项目, 免费私有仓库

+ `codeup.aliyun.com` (云效代码)

  免费私有仓库 (阿里云)

+ `atomgit.com`

  可存放开源项目

+ `gitee.com` (码云)

  可存放开源项目, 免费私有仓库
